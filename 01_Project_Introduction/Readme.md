<h1 align="center">🚖 GoodCabs Transportation Performance Analysis</h1>
<p align="center">Power BI | SQL | Python | Data Analytics</p>

👤 Author: PRANAV SHRIVASTAV

GoodCabs Transportation Performance Analysis is an end-to-end **data analytics and business intelligence project** focused on evaluating the operational performance of GoodCabs, a transportation service operating across tier-2 cities in India.

The objective of this project is to transform raw operational and passenger data into meaningful insights using **SQL, Python, Excel, and Power BI**. The analysis helps understand trip demand patterns, passenger behavior, revenue performance, and target achievement across different cities to support **data-driven business decisions**.

---

# 🎯 Project Objective

The main objectives of this project are:

- Analyze overall trip demand and revenue performance
- Understand passenger behavior and trip patterns
- Evaluate city-wise operational performance
- Compare actual performance with business targets
- Identify trends that impact transportation demand
- Build an interactive Power BI dashboard for business insights

---

# 🗂 Dataset Description

The dataset represents operational data from **GoodCabs transportation services** and contains multiple business entities used for analysis.

The dataset includes:

- **Trips Data** – Information about completed trips and ride details  
- **Passengers Data** – Passenger activity and travel behavior  
- **Cities Data** – City-level operational performance  
- **Targets Data** – Monthly performance targets assigned to each city  
- **Date Data** – Time dimension used for demand trend analysis  

These datasets were processed and analyzed using **SQL queries and Python notebooks** to generate meaningful insights.
